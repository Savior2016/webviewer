# momhand Skill - 物品管理

## 功能说明

momhand 是你的智能物品管理助手，帮你记录、查找、管理家中物品。

## 使用方法

### 📦 登记物品
```
我要放置一个物品
```
Friday 会询问你物品信息，你也可以直接一次性告诉我：

**完整信息示例：**
```
我要放置一个物品，感冒药，药品，用于治疗感冒头疼发烧，25 元，2024 年 1 月购买，有效期到 2028 年 6 月，放在电脑桌上的架子上
```

**简单示例：**
```
我把感冒药放客厅药箱了，30 块钱买的
```

你提供多少算多少，不强制所有字段。

### 物品信息字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 名称 | 物品名字 | 感冒药、笔记本电脑 |
| 类型 | 分类 | 药品、食品、电子产品、衣物、书籍、日用品、工具等 |
| 照片 | 物品图片 | 可发送图片给我 |
| 用途 | 用来做什么 | 治疗感冒、工作用 |
| 购买时间 | 什么时候买的 | 2024 年 1 月、上周 |
| 价格 | 多少钱 | 25 元、199 |
| 生产日期 | 生产时间 | 2024-01-01 |
| 保质期/有效期 | 何时过期 | 2028 年 6 月、2 年 |
| 位置 | 放在哪里 | 电脑桌上的架子、客厅药箱 |

### 🔍 查找物品
```
我的感冒药在哪里？
Find 我的笔记本电脑
```

### 📍 更新位置
```
我把感冒药移到抽屉里了
```

### ⏰ 查看即将过期物品
```
有什麽快过期了？
```

### 💡 需求推荐
```
我头疼，有什麽药？
我想泡茶，有什麽工具？
```
