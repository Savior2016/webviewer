# momhand - 物品管理系统

## 项目简介
momhand 是一个智能物品管理系统，帮助你记录、查找、管理家中物品。

## 功能特性

### 1. 📦 物品登记
告诉系统你要放置物品，会自动询问：
- 物品名称
- 类型（食品、日用品、电子产品、衣物、书籍等）
- 物品照片
- 用途
- 购买时间
- 价格
- 生产日期
- 保质期
- 放置位置

### 2. 🔍 物品查找
询问某个物品在哪里，系统告诉你位置。

### 3. ⏰ 到期提醒
定期检查，提醒即将过期的物品。

### 4. 📍 位置变更
物品移动位置时，更新记录。

### 5. 🌐 Web 展示
在浏览器查看所有物品，支持搜索、筛选。

### 6. 💡 需求推荐
产生需求时，系统推荐合适的物品。

## 项目结构

```
momhand/
├── skills/              # 技能脚本
│   └── item_manager.py  # 物品管理核心技能
├── web/                 # Web 展示页面
│   ├── index.html       # 物品列表页
│   └── api.py           # Web API
├── data/                # 数据存储
│   └── items.json       # 物品数据（SQLite/JSON）
└── README.md            # 本说明文件
```
