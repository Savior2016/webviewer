## 核心设计元素

### 1. 色彩系统
每个应用拥有独特的渐变色主题：

| 应用 | 主色 | 渐变 | 情感 |
|------|------|------|------|
| 首页 | 紫 + 粉 + 蓝 | `#0f0c29 → #302b63 → #24243e` | 神秘、科技感 |
| By Design | 蓝 + 青 | `#1e3a5f → #0f4c75 → #1b5e8b` | 专业、可靠 |
| Cherry Pick | 紫 + 粉 | `#4c1d95 → #6d28d9 → #7c3aed` | 温馨、活力 |
| Momhand | 绿 + 翠 | `#065f46 → #059669 → #10b981` | 自然、安心 |

### 2. 玻璃拟态 (Glassmorphism)
- 半透明卡片背景 `rgba(255, 255, 255, 0.95)`
- 背景模糊 `backdrop-filter: blur(20px)`
- 微妙边框 `border: 1px solid rgba(255, 255, 255, 0.2)`

### 3. 动画效果

#### 浮动动画 (Floating)
```css
@keyframes floating {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-10px); }
}
```
应用于：Emoji 图标、背景光斑

#### 滑入动画 (Slide In)
```css
@keyframes slideIn {
  from { opacity: 0; transform: translateY(20px); }
  to { opacity: 1; transform: translateY(0); }
}
```
应用于：卡片、内容块

#### 微光动画 (Shimmer)
```css
@keyframes shimmer {
  0% { background-position: -200% 0; }
  100% { background-position: 200% 0; }
}
```
应用于：装饰元素

### 4. 交互动效

#### 卡片悬停
- 上浮 `translateY(-12px)`
- 缩放 `scale(1.02)`
- 阴影增强 `box-shadow: 0 25px 50px`
- 过渡时间 `0.4s cubic-bezier(0.175, 0.885, 0.32, 1.275)`

#### 图标悬停
- 放大 `scale(1.15)`
- 旋转 `rotate(-5deg)`
- 阴影增强

#### 按钮悬停
- 上浮 `translateY(-2px)`
- 阴影增强
- 渐变背景

### 5. 字体系统
- **字体**: Inter (Google Fonts)
- **字重**: 400/500/600/700/800
- **标题**: Extrabold (800)
- **正文**: Regular (400) / Medium (500)

## 技术实现

### TailwindCSS 配置
```javascript
tailwind.config = {
  theme: {
    extend: {
      fontFamily: {
        sans: ['Inter', 'system-ui', 'sans-serif']
      }
    }
  }
}
```

### 通用样式类
- `.gradient-bg` - 渐变背景
- `.glass-card` - 玻璃拟态卡片
- `.floating` - 浮动动画
- `.slide-in` - 滑入动画
- `.btn-primary` - 主按钮样式

### 响应式设计
- 移动端优先
- 断点：md (768px), lg (1024px)
- 网格自适应：`grid-cols-2 md:grid-cols-4`

## 性能优化

1. **CDN 资源**: TailwindCSS + Google Fonts
2. **内联样式**: 关键 CSS 内联，减少请求
3. **动画优化**: 使用 `transform` 和 `opacity`，避免重排
4. **延迟加载**: 卡片交错显示，避免同时渲染
